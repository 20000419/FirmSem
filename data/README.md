# Public Data Slices

This directory exposes only the curated final evaluation records needed for
inspection and lightweight reuse.

| Directory | Scope |
|---|---|
| `development/` | Final obligation-level development gold and agreement report |
| `project_holdout/` | Final project-disjoint scored obligations and evaluation summary |
| `mechanism_holdout/` | Final incident-paired mechanism challenge and evaluation summary |
| `security_impact/` | Same-function counterfactual security-impact records |

The complete A/B/C returns, frozen prompts, model transcripts, exclusion
ledgers, and regeneration logs are distributed in the versioned archival ZIP,
not expanded into the GitHub source tree.
