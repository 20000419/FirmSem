# FirmSem v1.0.0

This release separates the maintainable public source tree from the complete
archival evidence package.

## Source tree

- Core GuardIR, obligation matching, cascade, metrics, and model-evaluation
  utilities.
- Curated final development, project-holdout, mechanism-holdout, and
  security-impact data.
- Executed 2+1 annotation protocol and a compact public regression suite.
- GCC/Clang security-impact replay source and frozen summaries.

## Archival asset

`FirmSem_complete_artifact_v1.0.0.zip` contains raw annotation returns, frozen
prompts and model transcripts, full analysis outputs, compiler replays, and
regeneration logs. Its manifest verifies 1,884 payload files.

Manuscript drafts, submission files, internal peer-review workflows, private
working notes, and failed diagnostic directories are intentionally excluded
from both the source tree and the archival asset.
