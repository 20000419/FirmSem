# FirmSem v2.0.0

This release accompanies the Route-B empirical-study revision. It separates
the maintainable public source tree from a compact frozen-evidence archive.

## Source tree

- Core GuardIR, obligation matching, cascade, metrics, and model-evaluation
  utilities.
- Curated development, project-disjoint, paired-incident, and
  security-impact data.
- Executed 2+1 annotation protocol and a compact public regression suite.
- GCC/Clang security-impact replay source and frozen summaries.

## Archival asset

`FirmSem_RouteB_Artifact_v2.0.0.zip` contains raw A/B annotation returns,
adjudications, final gold, frozen prompts and model transcripts, bounded
mechanism-attribution evidence, derived analyses, claim-to-evidence ledgers,
and audit scripts.

Manuscript drafts, submission files, internal peer-review workflows, private
working notes, and failed diagnostic directories are intentionally excluded
from both the source tree and the archival asset.

The compact archive also excludes documented multi-gigabyte project checkouts,
compiler build trees, and raw intermediate traces. It supports integrity
verification, software tests, frozen-evidence inspection, and deterministic
recomputation of selected derived analyses; it does not claim an end-to-end
replay of every acquisition stage.
